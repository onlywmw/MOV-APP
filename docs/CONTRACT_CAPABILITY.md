# CONTRACT_CAPABILITY — 能力边界表

> agent 计划时必须对照本表执行。**✅ 做到可上线标准，⚠️ 只能 demo 的明示，❌ 明说并给替代。**
> 能力变更（新增工具/打通权限/接入后端）必须同 commit 更新本表。这是"假菜单"原则的机器可读版。

## 〇、两个运行环境（一切判断的前提）

| 环境 | 是什么 | 能力来源 |
|------|--------|----------|
| **MOV 宿主内** | agent 通过 device.cmd 直接调本机 | 27 个原生能力 (CapabilityExecutor) |
| **打包 APK 内** | agent 产出的单文件 HTML 在独立 WebView 壳里跑 | 纯浏览器能力，**无任何原生桥** |

同一个词（如"拍照"）在两个环境里答案相反——计划涉及打包 APK 时，只能查"打包 APK 内"列。

## 一、✅ 能做 — 必须做到可上线完成度

| 能力 | 宿主内 | 打包 APK 内 | 完成度要求 |
|------|--------|------------|-----------|
| 单文件 HTML 工具/游戏/表单 | — | ✅ | 交互全部真绑定，视觉可交付 |
| HTML → 签名 APK（稳定包名覆盖升级） | — | ✅ | 安装、启动、杀进程重开数据在 |
| localStorage 持久化 | — | ✅ | 订单/记录类必用，禁止内存态假存 |
| 文件读写 + 版本快照 | ✅ | ❌（无桥） | 写前预览闸，快照可回滚 |
| 设备控制：手电/音量/亮度/震动/TTS/通知/剪贴板/电量/截屏/网络状态/系统信息/进程列表 | ✅ | ❌ | device.cmd 直接做，禁止 HTML 模拟 |
| 设备控制（需权限）：拍照 camera.photo / 定位 location.get / 通讯录 contacts.list / 短信 sms.recent / 拨打电话 | ✅（需用户授权） | ❌ | 权限被拒要有人话降级 |
| 应用启动 app.launch / 应用列表 / 模拟点击滑动 input.tap/swipe | ✅ | ❌ | — |
| Cron 定时任务 | ✅ | ❌ | 白名单 action，产出落房间+通知 |

**完成度通用标准（所有 ✅ 项）：承诺的功能全部真实现，禁止占位符、桩代码、"TODO"、"略"、图片用色块代替、按钮无响应。**

## 二、⚠️ 只能 demo — 必须明示"这是演示版"

| 场景 | 缺什么 | demo 边界 |
|------|--------|-----------|
| 多用户/账号体系界面（聊天、社交） | 后端 + 账号系统 | 单机界面 + 本地模拟数据，交付时明说"数据不出本机" |
| 财务/经营系统 | 真数据源 + 后端 | 手工录入 + localStorage 统计可以是真的；自动同步/对公接口是 demo |
| 协作/共享类（多人编辑、家庭共享） | 后端同步 | 单机版，明说"不能多人同时用" |

## 三、❌ 做不了 — 明说，给替代方案

| 需求 | 为什么做不了 | 替代方案 |
|------|-------------|----------|
| 打包 APK 内扫一扫/相机 | 打包器模板无 CAMERA 权限 + WebView 未开权限通道 | 路线图：打包器原生桥 v2；现在用 MOV 宿主内 camera.photo 代拍 |
| 打包 APK 内录音/语音识别/定位/蓝牙/NFC | 同上，无桥 | 路线图项 |
| 联网抓取（资讯、行情、研究） | http.get 工具未做 | 路线图 P2（域名白名单）；现在用户手动贴内容 |
| 真后端（账号、消息同步、多人实时） | agent 无服务器部署能力 | 交付 server.js + 部署说明（用户有云服务器可自部署）；或改单机需求 |
| 支付接口 | 资质 + 后端 | 展示收款码图片（烧烤摊模式） |
| 应用级推送（APK 后台收消息） | 需推送 SDK + 后端 | 宿主内 notification.post 本机通知 |
| 图片理解（拍照解题、票据识别） | AiClient 不支持图片消息 | 路线图 P2 |

## 四、执行规则（写进 prompt 的压缩版依据）

1. 计划前对照本表：需求落在 ✅ → 全部做掉；⚠️ → 计划卡必须标注"演示版"及边界；❌ → 不得闷头做，先说明并给替代。
2. 混合需求（部分 ✅ 部分 ❌）：✅ 部分全做，❌ 部分在计划卡里单独列出"此部分无法真实现"及替代。
3. 评审/验收以本表为准：✅ 项出现占位符 = 交付缺陷（返工）；⚠️ 项未明示演示边界 = 交付缺陷。
